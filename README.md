## API

`/` : 기본 화면

`GET : /login` : 로그인 화면

```json
{
  userid : "abc",
  password : "..",
  email : "abc@email.com",
  name : "name",
  phone : "010-1234-1234",
  postcode : "000-000",
  address1 : "",
  address2 : "",
  advcheck : "true",
}

```

`POST : /login` : 로그인 요청

`GET : /signup` : 회원가입 화면

`POST : /signup` : 회원가입 요청

`GET : /checkid` : 해당 아이디가 존재하는지 확인
